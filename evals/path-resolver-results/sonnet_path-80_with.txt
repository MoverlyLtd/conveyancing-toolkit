The PDTF path for damp or dry rot treatment documentation is:

**`/propertyPack/specialistIssues/dryRotEtcTreatment/attachments`**

This path covers treatment reports and guarantees for:
- Dry rot treatment
- Wet rot treatment  
- Damp proofing treatment
- Other timber preservation work

If there's also a **guarantee** for damp proofing treatment specifically, that would go to:
**`/propertyPack/guaranteesWarrantiesAndIndemnityInsurances/dampProofingTreatment/attachments`**

The specialist issues path is for the treatment report itself, while the guarantees path is for any warranty or guarantee documentation provided by the treatment company.
