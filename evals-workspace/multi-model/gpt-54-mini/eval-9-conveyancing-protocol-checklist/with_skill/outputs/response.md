If you’re at **exchange stage**, the key Law Society Conveyancing Protocol checks are:

## General obligations
- Verify the other side’s conveyancer identity. **(General, para (a))**
- Use the latest edition of the **Standard Conditions of Sale**. **(General, para (b))**
- Use the **Code for Completion by Post**. **(General, para (c))**
- Use current property forms and agreed documents. **(General, para (d))**
- Use the protocol exchange formulae when exchanging by telephone. **(General, para (e))**
- Only make essential amendments to the Standard Conditions. **(General, para (f))**
- Only vary the Code for Completion by Post if your client instructs you to do so. **(General, para (g))**
- If you’re acting for the buyer, only raise extra enquiries that are relevant and necessary. **(General, para (l))**

## Stage C: Exchange
- Exchange contracts using the agreed formula. **(Stage C, para 1)**
- Confirm exchange to your client immediately. **(Stage C, para 2)**
- If acting for the buyer, send the deposit to the seller’s conveyancer or confirm it is held as stakeholder. **(Stage C, para 3)**
- If acting for the seller, notify the estate agent of exchange. **(Stage C, para 4)**
- If acting for the buyer, submit the certificate of title to the lender. **(Stage C, para 5)**
- If acting for the seller, prepare the completion statement. **(Stage C, para 6)**
- Prepare the transfer deed for execution. **(Stage C, para 7)**
- If acting for the buyer, make sure buildings insurance is in place from exchange. **(Stage C, para 8)**
