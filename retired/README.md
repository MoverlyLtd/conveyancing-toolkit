# Retired Skills

These skills have been moved out of the active toolkit. They boosted benchmark scores (+20-30%) but their real-world use case was weak:

- **conveyancing-protocol-checklist** — Protocol compliance is audited after the fact, not followed step-by-step. The DE flags ARE the checklist.
- **ca-protocol-compliance** — Same reasoning. DE flags cover the substantive property-level checks.
- **cqs-practice-standards** — Firm-level practice management, not transaction-level intelligence. Out of scope for the DE.
- **clc-compliance-tracker** — Firm-level regulatory compliance, same as CQS.
- **property-law-reference** — Curated URL list. Models can search the web; a static reference list adds little value.

The gaps these skills covered in DE rule coverage are documented in `docs/de-coverage-audit.md` and have been briefed to the Argus team for inclusion in the diligence engine itself.

*Retired: 2026-04-03*
