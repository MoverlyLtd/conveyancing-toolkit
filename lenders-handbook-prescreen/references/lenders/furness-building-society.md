## 4.1 Is there a valuation report and if so, does the lender provide it?

A copy of the valuation report will be provided if a full valuation has been carried out. The mortgage offer will state where a full valuation report has not been obtained. The solicitor should rely on the report provided by the Society, not any report provided by the applicant and should contact the Society should a report not have been provided.
